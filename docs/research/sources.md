# Kisasa AI PM Pipeline — Research Sources

## 1. AI Foundations & Architecture

- **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)** — Production patterns for RAG, tool use, sub-agents, agentic loops, structured outputs (500+ examples)
- **[Anthropic Docs — Agentic Systems](https://docs.anthropic.com/en/docs/build-with-claude/agentic-systems)** — Recommended patterns: agentic loops, tool calling, stopping conditions
- **[Snowflake — AI Data Pipelines](https://www.snowflake.com/en/fundamentals/understanding-ai-data-pipelines/)** — Data pipeline fundamentals: ingestion, transformation, serving

## 2. Agent Frameworks

- **[Claude Agent SDK](https://github.com/anthropics/claude-agent-sdk)** — Native Claude agents with tool use
- **[LangGraph](https://langchain-ai.github.io/langgraph/)** — Stateful, graph-based agent workflows with branching and cycles
- **[CrewAI](https://www.crewai.com)** — Multi-agent collaboration for "team of AI agents" model

## 3. Product Management Methodology

- **[SVPG — Product Coaching and AI](https://www.svpg.com/product-coaching-and-ai/)** — Marty Cagan's updated take on AI in product discovery (Feb 2026)
- **[Shape Up (Basecamp)](https://basecamp.com/shapeup)** — Shaping, betting, and building cycles — pipeline stage model
- **[RICE Scoring Framework](https://www.intercom.com/blog/rice-simple-prioritization-for-product-managers/)** — Automated prioritization scoring for AI implementation

## 4. Technical Implementation

- **[Anthropic Cookbook — RAG](https://github.com/anthropics/anthropic-cookbook/tree/main/misc/retrieval_augmented_generation)** — Knowledge layer for past projects and client history
- **[Anthropic Cookbook — Tool Use](https://github.com/anthropics/anthropic-cookbook/tree/main/tool_use)** — Connecting Claude to Linear, Jira, GitHub APIs
- **[Anthropic Cookbook — Sub-agents](https://github.com/anthropics/anthropic-cookbook/tree/main/misc/sub_agents)** — Specialized agents (research, planning, QA)
- **[Vercel AI SDK](https://sdk.vercel.ai/docs)** — Streaming AI responses in Next.js dashboard UI

## 5. Pipeline & DevOps Patterns

- **[GitHub Actions](https://docs.github.com/en/actions)** — CI/CD automation for Build & Ship stage
- **[Prefect](https://www.prefect.io/)** — Python-native workflow orchestration for AI pipelines
- **[Dagster](https://dagster.io/)** — Alternative Python workflow orchestration
- **[OpenTelemetry](https://opentelemetry.io/)** — Observability and monitoring for production pipelines

## 6. Real-World References

- **[Linear Method](https://linear.app/method)** — Product development workflow design and UX inspiration
- **[Claude Code](https://github.com/anthropics/claude-code)** — Anthropic's AI-assisted dev tool — architecture reference
